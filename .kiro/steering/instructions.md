# Routeasy — Instruções do Projeto

## Sobre o App

Routeasy é um app mobile feito com React Native. Funciona como um assistente de rotas para entregadores ou qualquer pessoa que precise visitar múltiplos endereços.

O usuário cadastra os endereços que precisa visitar (ex: 5 endereços diferentes), e o app — integrado com as APIs do Google Cloud — calcula a melhor rota entre eles. O app também possui navegação integrada, sem precisar sair da aplicação.

O layout já está definido.

## Stack

- React Native (Expo)
- TypeScript (estrito)
- Tailwind (NativeWind)
- Google Cloud APIs (Maps, Routes, Directions)
- Bibliotecas adicionais serão definidas ao longo do desenvolvimento

## Ícones

- Usar exclusivamente `lucide-react-native` para todos os ícones do projeto
- Não usar `@expo/vector-icons` ou qualquer outra biblioteca de ícones

## Regras de Desenvolvimento

- Sempre responder em português
- Seguir orientação a testes: a cada mudança, escrever/rodar o teste correspondente
- Se o teste passar, seguimos para o próximo passo
- Desenvolvimento passo a passo — nunca dar múltiplas instruções de uma vez
- Os comandos do usuário sempre serão curtos e diretos, um passo por vez
- Nunca pular etapas sem validação
